CSS

There are 11 main CSS files in this theme.

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

Open the styles folder, you'll find all css files.

  • style.css - Theme common styles
  • bootstrap.min.css - bootstrap styles
  • font-awesome.min.css - Font Awesome stylesheet
  • animate.css - animation style
  • sm-clean.css - menu style
  • sm-core-css.css - menu style
  • owl.carousel.css - slider style
  • owl.theme.default.css - slider style
  • grt-youtube-popup.css - video style
  • lightbox.css - Modal style
  • responsive.css - responsive style

  3. popper.min.js - main custom jQuery
  4. modernizr.min.js - main custom jQuery
  5. TweenMax.min.js - ScrollToPlugin jQuery
  6. ScrollToPlugin.min.js - ScrollToPlugin jQuery
  7. Isotope.js - projects slider
  8. MagnificPopup.js - projects slider
  9. jquery.smartmenus.min.js - menu js
  10. owl.carousel.js - slider js
  11. cascade-slider.js - slider js
  12. wow.min.js - wow js
  13. jQuery.scrollSpeed.js - scrollSpeed js
  14. grt-youtube-popup.js - video js
  15. lightbox.js - Modal js
  16. custom.js - custom js
